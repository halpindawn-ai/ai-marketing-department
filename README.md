## Why this exists

Most AI-for-marketers content is tool roundups or theory. I haven't seen anyone writing about the structural layer, the work you do before the build becomes possible. The Substack is where I write about that layer. This repo is where the artifacts live.

If you're building AI into a marketing function and don't know where to start, the role spec template is the place to start. It's the same scaffolding behind every employee on my team.

## How to use these

Each template stands alone. The fastest path: download the one you want, drop it into Claude, and ask *"what do you need from me to build this skill for [your company]?"* Claude will interview you for the specifics rather than make you fill in brackets by hand.

For the **role spec template**, work through the eleven sections in order. Most of mine took 2 to 3 drafts. Write the spec before you draw the org chart, because the handoffs in Section 7 are where the cross-employee structure emerges. Use Section 8 (gap analysis) as your build roadmap.

For the **shared brain references** (writing standards, ICP, strategic context, brand guidelines), build them once and load them into every employee's spec. They're the substrate everything else reads from.

For the **build-order template**, pick the first layer, build it, run it for one or two cadences, and let the output tell you what to build next. The order matters more than the destination.
